# live-withdrawal-all

这是我的第一个前端学习项目。

## 项目介绍
这是一个用户管理提现的页面，可以看到用户的个人信息、提现记录等。

## 用到的技术
- Vue3: 用于页面和组件
- Vite：用于项目运行
- Git & GitHub: 代码上传

## 目录说明
- src/assets/images: 存放图片
- src/components: 公共组件，比如菜单、列表、分页等
- src/view：项目的主要页面，比如提现和直播记录等
- App.vue: 项目的根组件
- main.js: 项目入口文件

## 运行方式
1. 安装依赖: npm install
2. 启动项目: npm run dev
3. 访问地址: http://localhost:5173

- 学会使用 Git 上传代码
